# Project Brief: Endurain

## Core Requirements & Goals

Endurain is a self-hosted fitness tracking service, similar to Strava, but focused on user privacy, data ownership, and customization.

**Key Goals:**

*   Provide a robust platform for users to track their fitness activities.
*   Ensure users have full control over their data.
*   Offer integrations with popular services like Strava and Garmin Connect.
*   Support manual activity uploads via `.gpx` and `.fit` files.
*   Maintain a user-friendly interface.
*   Be easily deployable via Docker.

## Scope

The project encompasses:

*   **Backend:** API for managing users, activities, gear, health data, integrations, etc.
*   **Frontend:** Web interface for users to view and manage their data.
*   **Database:** Storage for all user and application data.
*   **Integrations:** Mechanisms to connect with external services (Strava, Garmin).
*   **File Processing:** Handling uploads and parsing of `.gpx` and `.fit` files.
*   **Deployment:** Docker-based deployment setup.

This document serves as the foundational source of truth for the project's scope and direction.
